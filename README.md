# AlgoEat

AI 기반 개인 맞춤형 메뉴 추천 및 식단 관리 서비스

## Tech Stack

- Frontend : React
- Backend : Spring Boot
- AI Server : Flask
- Database : MySQL
- Cloud : AWS EC2
