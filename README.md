# AlgoEat

AI 기반 음식 추천 서비스

## Tech Stack

- Frontend : React
- Backend : Spring Boot
- AI Server : Flask
- Database : MySQL
- Cloud : AWS EC2

## Project Structure

frontend/   React UI
backend/    Spring Boot API
ai-server/  Flask AI Recommendation Server
